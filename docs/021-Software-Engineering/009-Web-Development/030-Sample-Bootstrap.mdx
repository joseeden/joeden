# Rendering HTML

Note that Google Drive does not allow embedding files in an <iframe>, even if the file is publicly shared. The Google Drive viewer page blocks embedding for security reasons.

```html
import React from "react";

<iframe
  src="https://drive.google.com/file/d/194ILry_ILrOUqMjDxNGE-uqaeA1IBCAA/view"
  width="100%"
  height="600px"
  style={{ border: "1px solid #ccc" }}
></iframe>
```

<iframe
  src="https://drive.google.com/file/d/194ILry_ILrOUqMjDxNGE-uqaeA1IBCAA/view"
  width="100%"
  height="600px"
  style={{ border: "1px solid #ccc" }}
></iframe>

Instead of the **viewer link**, you need to use the **direct download** link format:

```bash
https://drive.google.com/uc?export=download&id=FILE_ID
```

Example:

```html
import React from "react";

<iframe
  src="https://drive.google.com/uc?export=download&id=194ILry_ILrOUqMjDxNGE-uqaeA1IBCAA"
  width="100%"
  height="600px"
  style={{ border: "1px solid #ccc" }}
></iframe>
```
